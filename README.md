# Protein structure analysis

Calculation of protein features. 

Jupyter notebook with function to calculate various protein features from pdb files in the protein data bank. 
* protein length
* surface area 
* fractions of secondary structures
* charge at pH 7 
* Isoelectric point 
* aromaticity 
* flexibility 
* molecular weight 
* instability 
* average b-factor
* Ramachandran plot
